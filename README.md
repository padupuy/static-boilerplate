# Static Boilerplate
Cet outil est conçu pour faciliter la création de sites web simples grâce au confort des outils modernes.

Ce boilerplate est inspiré de [blendid](https://github.com/vigetlabs/blendid)


## Sans
- jquery
- sass
- less
- bootstrap
